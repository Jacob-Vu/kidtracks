# KidsTrack Mobile

This folder contains the Flutter mobile client for Android and iOS.

## Status

The Flutter CLI is not available in the current workspace shell, so this repo includes:

- a build-ready Dart source structure under `lib/`
- app routing, Riverpod state, Firebase services, and feature screens
- ARB localization files for English and Vietnamese
- setup instructions for completing FlutterFire/platform generation locally

## Next local steps

1. Install Flutter and verify with `flutter doctor`.
2. From `mobile/`, run `flutter create .`.
3. Add Firebase config with FlutterFire:
   - `flutterfire configure`
4. Ensure callable functions use region `asia-southeast1`.
5. Run:
   - `flutter pub get`
   - `flutter gen-l10n`
   - `flutter run --dart-define=FIREBASE_API_KEY=your_web_api_key`

## Expected generated files

The app imports `firebase_options.dart`, which should be generated by FlutterFire in `lib/firebase_options.dart`.

Kid account creation uses the same Identity Toolkit REST flow as the web app, so pass the Firebase web API key with `--dart-define=FIREBASE_API_KEY=...`.
